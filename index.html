Chatbot for CheckIn API

Bachelorprosjekt for CheckIn AS: En RAG-basert chatbot som gir intelligente svar om CheckIns GraphQL API ved hjelp av AI og dokumentasjonssøk.

Om Checkin AS

Teknisk Ekspertise

Vi har utviklet en fullstendig RAG-pipeline med Streamlit, ChromaDB og valgfri OpenAI-integrasjon for å levere presise svar om CheckIns API.

Systemdesign

Chatboten er bygget med en modulær arkitektur: Streamlit UI, RAG-lag med vektorbasert søk, og støtte for både lokal og skybasert kjøring via Docker.

Problemløsning

Vi analyserer GraphQL-skjemaer og API-dokumentasjon for å gi utviklere raske og nøyaktige svar på tekniske spørsmål.

Kodekvalitet

Prosjektet inkluderer automatiserte tester, kontinuerlig integrasjon og grundig dokumentasjon for enkel vedlikehold.

Prosess

Med agile arbeidsmetoder, Git-versjonskontroll og strukturerte milepæler sikrer vi effektiv prosjektutvikling.

Teknologisk Fornyelse

Vi bruker moderne teknologier som RAG (Retrieval-Augmented Generation), vektorembeddings, ChromaDB og Docker for containerisert deployment.
